# EDUCATECH – Responsive Educational Website

## 📖 About
EDUCATECH is a responsive educational website built using HTML, CSS, and JavaScript. It provides an engaging learning experience with dedicated course pages for HTML, CSS, and JavaScript.

## ✨ Features
- Responsive Design
- Home Page
- About Section
- HTML Course Page
- CSS Course Page
- JavaScript Course Page
- Contact Form
- Mobile-Friendly Layout
- Modern UI Design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
EDUCATECH/
│── index.html
│── html-course.html
│── css-course.html
│── javascript-course.html
│── css/
│── js/
└── images/
```

## 🚀 How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
